HTML Y Unicode

Para utilizar un caracter particular de un idioma utiliza el codigo UNICODE
correspondiente precedido de "&#" y seguido de ";" (punto y coma), por ejemplo, el código
21512, produce esto: 合

Ejemplos:

Código Caracter Descripción
&#916; Δ Letra griega Delta
&#12354; あ Letra A Hiragana (japonés)
&#21494; 叶 Idiogarma "Hoja" en Chino Simplificado
&#33865; 葉 Idiogarma "Hoja" en Chino Tradicional

Pero también puedes mostrar símbolos matemáticos:

8734= ∞(infinito) 8721 = ∑ (sumatoria) 8736= ∠ (Angulo) 188 = ¼ (Fracción)
